UI Development

   

This is a demo project to learn more about Angular. This Angular project has created few components and one service. This project will talk about router, httprequest(GET, POST, DELETE), form, animation.

This Demo project created using some technologies like Angular, Typescript, HTML5, CSS3, Bootstrap4.

 
 

Angular Demosss Detailes:

Components and Services
Header Component

In this component Router is created which has few routes like home, about, contact, feedback, signup and login.

Navigation bar is responsive with toggle button and signup/login are showing in model box of bootstrap.
Home Component

Here carousel (image slide) is created with the help of third party ngx bootstrap.

In this componet whole project details has been described about Components and services.
Contact Component

Pending work:Need to add address and map.
Feedback Component

Pending work: Will add forms to send the feedback to my mail
Register Component Created a form with submit button and with the help of local storage. Need to add validation.
Login Component

Created a form. To login it takes local storage value and after login it redirect to home component
Httprequest Component

Here getting the data using GET method from JSONPlaceholder which is a free online REST API. You can Post the data using POST method to the list. You can delete the post from postes.

Keyframe Component

In this componet you will find some animation using few classes of animation. Animation is work on the page refresh, on button click and on mousehover
Stagger Component

Created some animation using query and stagger class of animation. Here animation take place on page refreash and on adding items and removing items.
vertical-nav Component

Created a vertical menu and on clicking the menu items content is displayed on same page.
Post Service

Created a service for Httprequest componet. Where I have kept some methods like get, post and delete
